How It Works

① Fetch

Every few minutes, a Python script fetches the latest articles from 15+ trusted RSS feeds worldwide. Stories are filtered for quality, relevance, and to avoid duplicates.

② AI Enhance

Each article is processed by AI to generate key points, FAQs, sentiment analysis, improved headlines, and structured metadata for better readability.

③ Publish

Processed stories are committed to GitHub automatically. GitHub Actions rebuilds the site via Jekyll and GitHub Pages in minutes. No manual intervention required.

Technology

GlobalBR News is built on a fully open-source, zero-cost stack:

Jekyll GitHub Pages GitHub Actions Python 3.11 feedparser Mistral AI Bootstrap 5 edge-tts FFmpeg YouTube Data API v3 Pillow

Automation Pipeline

  • Cron job runs every hour via GitHub Actions
  • RSS feeds fetched from 30+ global sources
  • Duplicate detection prevents republishing
  • Mistral AI enriches each story (title, summary, FAQ, key points)
  • Quality gate (0–10) drops thin posts before publish
  • Jekyll builds the static site; GitHub Pages serves it

What the AI does (and doesn't)

  • Does: rewrites the headline for clarity + SEO
  • Does: writes the enhanced summary, key points, FAQ, tl;dr
  • Does: tags entities and classifies sentiment
  • Doesn't: generate new facts — only restates what the source reported
  • Doesn't: remove the link back to the original source
  • Doesn't: replace the journalist who did the actual reporting
