How It Works

① Collect

Every hour, a Python script fetches the latest articles from 15+ trusted RSS feeds. Each story is filtered for quality and relevance before publishing.

② Publish

Approved stories are automatically formatted and committed to GitHub. GitHub Actions rebuilds the site in minutes via GitHub Pages.

③ Broadcast

Each story is also narrated with AI text-to-speech and published as a video on our YouTube channel.
