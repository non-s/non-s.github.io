TikTok OAuth callback

This page receives the redirect from TikTok during the one-time authorization flow for the Wild Brief publishing bot. It does not store anything in the browser and does not transmit anything to any third party — the authorization code is shown below for the operator to paste into the token-exchange script run locally.
