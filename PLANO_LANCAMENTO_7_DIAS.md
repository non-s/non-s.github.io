# 📅 Plano de Lançamento - 7 Dias (AUTOMATIZADO)

## 🎯 Estratégia Geral

**Objetivo:** Estabelecer identidade do canal para o algoritmo do YouTube  
**Frequência:** 3 Shorts/dia + 2 Horizontais/semana + 1 Live/semana  
**Foco:** Fofura + Relaxamento + Jazz  
**Execução:** 100% automatizada via GitHub Actions ⚙️

---

## 🤖 Automação Configurada

### ✅ Workflows Ativos:

| Workflow | Frequência | Horário (BRT) | Arquivo |
|----------|-----------|---------------|---------|
| **Shorts** | 1x/hora (24x/dia) | Todo horário cheio | `.github/workflows/pata-jazz-shorts.yml` |
| **Horizontais** | 1x/dia | 10:00 | `.github/workflows/pata-jazz-horizontal.yml` |
| **Lives** | 1x/semana | Quarta 19:00 (6h, 720p) | `.github/workflows/pata-jazz-youtube-live.yml` |

### 🚀 Como Funciona:

1. **GitHub Actions** dispara nos horários configurados
2. **Scripts Python** geram vídeo automaticamente
3. **SEO 2.0** cria título, descrição, hashtags
4. **Thumbnail** gerada com frame real do vídeo
5. **Upload automático** para YouTube via API
6. **Zero intervenção manual** necessária!

## 📈 Métricas para Acompanhar (Primeira Semana)

### 🎯 KPIs Principais:
- **CTR (Click-Through Rate):** Meta > 5%
- **Retenção:** Meta > 50% (Shorts), > 40% (Horizontais)
- **Engajamento:** Likes/Views > 3%
- **Inscritos:** Meta 50-100 na primeira semana

### 📊 Onde verificar:
1. **YouTube Studio** → Analytics
2. **Primeiras 48h** são críticas
3. **Shorts:** velocidade de views importa mais que retenção

### 🔄 Ajustes:
- **CTR baixo (<3%):** Mudar thumbnails/títulos
- **Retenção baixa:** Melhorar hooks nos primeiros 3s
- **Poucos inscritos:** Adicionar CTA mais forte

## 🎨 Dicas de Thumbnail (CRÍTICO)

### ✅ DO:
- Usar **frames reais do vídeo** (já implementado!)
- **Rosto do animal** em destaque (olhos grandes = fofura)
- **Cores vibrantes** (saturação +20-30%)
- **Texto curto** (2-3 palavras no máximo)
- **Emoji relevante**

### ❌ DON'T:
- Muito texto (ilegível em mobile)
- Imagens genéricas de banco
- Cores muito escuras
- Múltiplos animais (confunde)
