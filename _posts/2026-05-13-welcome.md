Bem-vindo ao **TechBR News**! 🎉

Este é o seu novo portal de notícias de tecnologia, construído com [Jekyll](https://jekyllrb.com) e hospedado gratuitamente no **GitHub Pages** — com atualizações automáticas três vezes por dia.

<!--more-->

## O que você vai encontrar aqui

O TechBR News foi criado para reunir as melhores notícias de tecnologia do Brasil em um único lugar, com curadoria automática dos principais veículos:

- **Tecnologia geral** — novidades do mundo tech
- **Gadgets** — smartphones, notebooks, wearables e mais
- **Inteligência Artificial** — IA, machine learning e automação
- **Startups** — empreendedorismo e inovação brasileira
- **Segurança Digital** — privacidade, cibersegurança e proteção de dados
- **Mobile** — apps, sistemas operacionais e dispositivos móveis

## Como o blog funciona

Este blog utiliza automação via **GitHub Actions** para se manter atualizado sem nenhum esforço manual. Três vezes por dia (8h, 12h e 18h, horário de Brasília), um script Python:

1. Lê os feeds RSS dos principais portais de tecnologia brasileiros
2. Identifica as notícias mais recentes
3. Cria posts formatados automaticamente
4. Publica tudo via commit no GitHub

Todo o conteúdo é **curado** — ou seja, os posts trazem um resumo + link direto para a fonte original, respeitando os direitos autorais dos veículos.

## Tecnologias utilizadas

| Tecnologia | Função |
|---|---|
| Jekyll | Gerador de site estático |
| GitHub Pages | Hospedagem gratuita |
| GitHub Actions | Automação de publicação |
| Bootstrap 5 | Interface responsiva |
| Python + feedparser | Coleta de notícias RSS |
| jekyll-seo-tag | Otimização SEO automática |
